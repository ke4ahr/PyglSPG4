## Notes

- All public APIs are exposed via the pyglspg4.api package
- Core SGP-4 logic is isolated in pyglspg4.sgp4
- Numerical math is backend-agnostic and routed through pyglspg4.backend
- No global mutable state is used anywhere in the tree
- Tests mirror package structure for clarity and coverage
- Documentation is kept in docs for GitHub and ReadTheDocs compatibility
